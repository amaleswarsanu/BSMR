# BSMR Group Website

A modern, high-performance website for BSMR Group featuring professional design, interactive elements, and responsive layout.

## Features
- **Modern UI**: Sleek, professional aesthetic with hyper-modern design elements.
- **Interactive Animations**: Dynamic hero backgrounds and smooth transitions.
- **Responsive Design**: Fully optimized for desktops and mobile devices.
- **Multi-page Structure**: Comprehensive sections for Strategy, Sustainability, Vision, and Careers.

## Project Structure
- `index.html`: Homepage
- `about.html`: Company overview
- `vision.html`: Vision and values
- `strategy.html`: Strategic overview
- `sustainability.html`: ESG and sustainability initiatives
- `careers.html`: Job opportunities
- `contact.html`: Contact information
- `styles.css`: Global styling and design system
- `script.js`: Interactive elements and animations

## How to Run
Simply open `index.html` in any modern web browser.
